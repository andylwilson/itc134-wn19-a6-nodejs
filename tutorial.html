Hello World! Tutorial

So you want to get your feet wet with Node.js!

Here is a walkthough of the most basic operation: displaying "Hello World" on a page.

Getting Started

There are a few things you'll need to do before anything will work properly:

  • Head to nodejs.org to download the node.js files to your computer
  • Or be working in a development environment like Cloud9 where node.js is already set up

Let's Begin

Your program will start off with calling a required module and saving it to a variable. 


        var http = require('http);

The http module allows node.js to transfer data over http and contains many other useful functions.

Setup the function

One of the functions in the http module that we will be using in the program is createServer(). Which does exactly as the name implies:


        http.createServer(
            function(req,res)
                {
                res.writeHead(200, {'Content-Type':'text/plain'});
                res.end('Hello World\n');
                }
        ).listen(8080);

The writeHead() fuction sets up some parameters for the web browser:


        {res.writeHead(200, {'Content-Type':'text/plain'});}
  • 200 is a status code which tells the browser that everything is OK
  • Content-Type tells the browser what format the information will be in - in our case plain text

At the end of the createServer() function we tell the server which port to listen to when running our program:


        ).listen(8080);

How to Run

Once you have gotten all the code written in your program now it's time for the fun part! Testing it out live!

Your final code should look like this and be saved with an easy to remember name like "hello.js":


        var http = require('http');

        http.createServer(function (request, response) 
        {
          response.writeHead(200, {'Content-Type': 'text/plain'});
          response.end('Hello World\n');
        }).listen(8080);

In Cloud9 running this file is a simple matter of selecting the Run option from the menu and waiting for it to generate a url to click on. If everything is working properly, when you click on this link the words "Hello World" will be displayed on your browser page.

If you are set up to work on your own computer you will need to run the file from the command line with the command: 


        node hello.js

You should see 'Server started' and then you can open up a new tab in the browser and type in http://localhost:8080 and "Hello World" should be displayed on the screen.
